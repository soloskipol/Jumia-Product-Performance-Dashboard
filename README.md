# Jumia-Product-Performance-Dashboard

**Project Overview**

This project analyzes Jumia product data to identify trends in pricing, discounts, customer reviews, and ratings. The dashboard was developed in Microsoft Excel using Pivot Tables, Pivot Charts, Slicers, and Conditional Formatting.

**Objectives**
Analyze product ratings and reviews.
Evaluate the impact of discounts on customer engagement.
Compare product performance across discount categories.
Create an interactive dashboard for business insights.

**Tools Used**
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting

**Dashboard Features**
Total Products KPI
Average Rating KPI
Average Discount KPI
Total Reviews KPI
Rating Distribution Analysis
Review vs Rating Trend Analysis
Discount vs Review Trend Analysis
Product Performance Tables
Interactive Filtering using Slicers

**Key Insights**
Average product rating is 3.9 out of 5.
Average discount across products is 37%.
Medium-discount products have the highest average rating.
Products with higher discounts do not necessarily receive more reviews
